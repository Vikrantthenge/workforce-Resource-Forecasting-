## ðŸ§® Python Forecasting Pipeline (Prophet + ARIMA)

### Steps:
1. Load hourly data  
2. Aggregate to daily series  
3. Fit Prophet  
4. Fit ARIMA  
5. Compare MAE / RMSE  
6. Export unified forecast CSV  

---

### Prophet Example Code

```python
from prophet import Prophet
import pandas as pd

df = pd.read_csv("workforce_forecast_dataset.csv", parse_dates=['Timestamp'])

daily = df.groupby(df['Timestamp'].dt.date)['ActualVolume'].sum().reset_index()
daily.columns = ['ds', 'y']

model = Prophet(daily_seasonality=True, weekly_seasonality=True)
model.fit(daily)

future = model.make_future_dataframe(periods=30)
forecast = model.predict(future)
forecast.to_csv("forecast_output_30days.csv", index=False)
